THE DEVELOPMENT PROCESS

This project was assigned for students to demonstrate their accumulated knowledge from the CSS and web development course, SE133.
This website is a live representation of the skills I have gained throughout my education as a Software Engineer.
Below you will see my creative process in developing this very site, "Tangled Effort".
You can access the GitHub Repository here.

Inspiration

Before designing anything, I look for a general piece to use as a reference for inspiration.
For my portfolio, I wanted to implement natural warm tones utilized throughout minimalist home designs.

Take It a Step Further

After finding a good reference, I will begin sampling various color schemes until I find a combination I can be confident in.

Putting The Pieces Together

While I choose which color schemes to implement, I use Milanote to sort my thoughts and label what each color is assigned to which content.
At this point, I am now also deciding on which font faces to use as well.
